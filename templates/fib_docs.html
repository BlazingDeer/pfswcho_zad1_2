Fibonacci Calculator

Documentation



Aplikacja składa się z trzech kontenerów: frontend, API oraz bazy danych. Frontend przedstawia interfejs użytownika oraz komunikuje się z API. API oblicza wartość ciągu Fibonacciego na podstawie liczby otrzymanej z konteneru frontend i komunikuje się z postresql BD w celu zapisania wartości (Metoda POST). W przypadku metody GET, API żąda zwrócenia ostanich 10 zapisanych wartości z bazy danych.

Frontend został napisany przy użyciu frameworka Django w języku python a komunikacja pomiędzy frontendem a API jest zrealizowana poprzez moduł Requests.


API również zostało wykonane w języku python przy pomocy Django oraz Django Rest Framework. Zachowuje się w nim model bazy danych oraz kod obliczający wartość ciągu. Serializer przekłada model na obiekt json, natomiast views pozwalają na komunikację API z frontendem. Połączenie z DB jest automatycznie wykonywane przez Django przy podaniu odpowiednich ustawień dla projektu.


Baza danych jest kontenerem Dockera i oprócz hasła i nazwy użytkownika nic nie zostało zmienione.


Wszystkie kontenery są połączone poprzez docker-compose.

wszystkie kontenery są uruchamiane poprzez polecenie:

docker-compose up -d --build
